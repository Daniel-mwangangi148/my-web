Module 1: Introduction to Java

Java is a high-level, class-based, object-oriented programming language. It is widely used due to its portability, performance, and security features.

Advantages of Java

  • Platform Independent (Write Once, Run Anywhere)
  • Object-Oriented
  • Secure
  • Robust and Reliable

How Java Achieves Cross-Platform

Java uses the Java Virtual Machine (JVM) to run bytecode, which allows the same program to run on any device with a JVM installed.

Installing Java and Setting Up

  • Download JDK from Oracle's website
  • Install and set environment variables (JAVA_HOME and PATH)
  • Use a code editor like VSCode or IntelliJ

First Java Program

// This is a simple Java program
public class HelloWorld {
    public static void main(String[] args) {
        // Prints text to the screen
        System.out.println("Hello, world!");
    }
}

Module 2: Variables and Data Types

Variables store data and are the building blocks of any program.

Common Data Types:

  • int - integers
  • double - decimal numbers
  • char - single characters
  • boolean - true or false
  • String - text

Rules to Consider When Naming a Variable

  • Must start with a letter, $, or _
  • Cannot use spaces
  • Should be meaningful
  • Java is case-sensitive
// Declaring variables
int age = 25;
double price = 19.99;
char grade = 'A';
boolean isJavaFun = true;
String name = "Alice";

Module 3: Control Structures

Control structures let you control the flow of execution in your program using conditions and loops.

1. If-Else Statement

This checks a condition and runs different code based on whether it’s true or false.

// Declare a variable to hold age
int age = 18;

// Check if age is 18 or more
if (age >= 18) {
    // If true, this block will execute
    System.out.println("You are an adult.");
} else {
    // If false, this block will execute instead
    System.out.println("You are not an adult.");
}

2. Switch Statement

Use switch to handle multiple conditions based on the same variable.

// Declare a grade variable
char grade = 'B';

// Switch checks the value of grade
switch (grade) {
    case 'A':
        // If grade is A
        System.out.println("Excellent!");
        break;
    case 'B':
        // If grade is B
        System.out.println("Good job!");
        break;
    case 'C':
        // If grade is C
        System.out.println("You passed.");
        break;
    default:
        // If grade is none of the above
        System.out.println("Grade not recognized.");
}

3. While Loop

Repeats code while a condition is true.

// Start counting from 1
int i = 1;

// Run the loop as long as i is less than or equal to 5
while (i <= 5) {
    // Print the current value of i
    System.out.println("Count: " + i);

    // Increase i by 1 to avoid infinite loop
    i++;
}

4. For Loop

Used when you know how many times to run the loop.

// Loop from 1 to 5
for (int i = 1; i <= 5; i++) {
    // Print the step number
    System.out.println("Step: " + i);
}

5. Do-While Loop

Runs the code at least once, then checks the condition.

// Start i at 1
int i = 1;

// Run the loop at least once before checking the condition
do {
    // Print the current number
    System.out.println("Iteration: " + i);

    // Increase i
    i++;
} while (i <= 5); // Repeat while i is less than or equal to 5

6. Nested Loops (Loops inside loops)

You can place one loop inside another to perform repeated tasks within repeated tasks. This is useful for working with patterns, tables, or grids.

Example: Print a simple star pattern (a square of stars):

// Outer loop controls the rows
for (int row = 1; row <= 4; row++) {
    // Inner loop controls the columns (stars in each row)
    for (int col = 1; col <= 4; col++) {
        System.out.print("* ");  // Print star and space, stay on the same line
    }
    System.out.println();  // Move to next line after inner loop finishes
}

Output:


* * * * 
* * * * 
* * * * 
* * * *

Example 2: Print a right-angled triangle star pattern:

// Outer loop controls the rows
for (int row = 1; row <= 5; row++) {
    // Inner loop prints stars equal to current row number
    for (int col = 1; col <= row; col++) {
        System.out.print("* ");
    }
    System.out.println(); // Move to the next line
}

Output:


* 
* * 
* * * 
* * * * 
* * * * *

Tip: Nested loops can be used for more complex tasks like working with 2D arrays or creating tables.

Module 4: Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

What is an Array?

An array is a container that holds a fixed number of values of the same type. Each value is called an element, and each element has an index starting from 0.

Declaring and Initializing Arrays

// Declare an array of integers with 5 elements
int[] numbers = new int[5];

// Initialize an array with values
String[] fruits = {"Apple", "Banana", "Cherry"};

Accessing Array Elements

You access array elements using their index inside square brackets []. Remember, arrays start at index 0.

// Access first element
System.out.println(fruits[0]);  // Prints "Apple"

// Change the value of the second element
fruits[1] = "Blueberry";
System.out.println(fruits[1]);  // Prints "Blueberry"

Looping Through Arrays

You often use loops to go through each element of an array.

// Using a for loop to print all fruits
for (int i = 0; i < fruits.length; i++) {
    System.out.println(fruits[i]);
}

Important Points

  • Array size is fixed after creation; you cannot change it later.
  • Trying to access an index outside the array length causes an ArrayIndexOutOfBoundsException.

Module 5: Object-Oriented Programming (OOP) Basics

OOP helps you organize code into reusable objects. It focuses on creating classes, objects, and using real-world concepts like inheritance, encapsulation, and polymorphism.

1. Classes and Objects

A class is a blueprint. An object is an instance (real version) of that blueprint.

// Define a class called Car
public class Car {
    // Property (field) of the class
    String color;

    // Method to display car color
    void displayColor() {
        System.out.println("The car color is: " + color);
    }
}

// Main class to run the program
public class Main {
    public static void main(String[] args) {
        // Create an object (instance) of Car
        Car myCar = new Car();

        // Set the color of the car
        myCar.color = "Red";

        // Call the method to display the color
        myCar.displayColor();
    }
}

2. Encapsulation

Encapsulation hides the data and only exposes what is necessary. You use private fields and getters / setters.

// Define a class with encapsulated fields
public class Person {
    // Private field - cannot be accessed directly from outside
    private String name;

    // Setter method - to set value
    public void setName(String newName) {
        name = newName;
    }

    // Getter method - to get value
    public String getName() {
        return name;
    }
}

public class Main {
    public static void main(String[] args) {
        // Create a Person object
        Person p = new Person();

        // Set the name using the setter
        p.setName("Kalos");

        // Get and print the name using the getter
        System.out.println("Name: " + p.getName());
    }
}

3. Inheritance

Inheritance allows one class to reuse the properties of another. We use extends keyword.

// Parent class
class Animal {
    void speak() {
        System.out.println("Animal speaks");
    }
}

// Child class that inherits from Animal
class Dog extends Animal {
    // Additional method
    void bark() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        // Create a Dog object
        Dog d = new Dog();

        // Call inherited method
        d.speak();

        // Call its own method
        d.bark();
    }
}

4. Polymorphism

Polymorphism means many forms. Java lets you override methods to behave differently based on the object.

// Parent class
class Animal {
    void makeSound() {
        System.out.println("Some generic sound");
    }
}

// Child class with overridden method
class Cat extends Animal {
    void makeSound() {
        System.out.println("Meow");
    }
}

public class Main {
    public static void main(String[] args) {
        // Polymorphism: Animal reference, Cat object
        Animal a = new Cat();

        // Calls the overridden method
        a.makeSound();
    }
}

5. Constructors

A constructor initializes an object when it's created.

// Class with a constructor
public class Student {
    String name;

    // Constructor
    Student(String studentName) {
        name = studentName;
    }

    void sayHello() {
        System.out.println("Hi, I'm " + name);
    }
}

public class Main {
    public static void main(String[] args) {
        // Create object with constructor
        Student s = new Student("Don Kalos");

        // Call method
        s.sayHello();
    }
}
